# Sonificació de la posició dels planetes al cel

Aquest projecte agafa la posició dels planetes relatives a una posició determinada en un rang temporal especificat i crea una sonificació. <br/>
Obtenció de dades utilitzant la llibreria <a href="https://rhodesmill.org/pyephem/index.html">pyEphem</a> i sonificació generada utilitzant <a href="https://cycling74.com/products/max">Max/MSP</a>. <br/>

Es poden trobar demos a la [carpeta corresponent](/demos).

# Execució
Instal·lar [Jupyter Notebooks](https://jupyter.org/).

Obrir [main.ipynb](main.ipynb) i executar els blocs en l'ordre corresponent.
